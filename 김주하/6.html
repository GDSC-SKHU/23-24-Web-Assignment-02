산술 연산자

연산자 예 설명
+ c = a + b a 와 b 의 합을 c 에 저장, 덧셈 연산
- c = a - b a 와 b 의 차를 c 에 저장, 뺄셈 연산
* c = a * b a 와 b 의 곱을 c 에 저장, 곱셈 연산
** c = a ** b a 의 값을 b만큼 거듭제곱한 결과를 c에 대입
/ c = a / b a 와 b 로 나눈 몫을 c 에 저장, 나눗셈 연산
% c = a % b a 와 b 로 나눈 나머지를 c 에 저장

대입 연산자

연산자 예 설명
= c = a a 의 값을 변수 c에 대입
+= c += a c 의 값에서 a 를 더한 결과를 c에 대입, 덧셈 연산
-= c -= a c 의 값에서 a 를 뺀 결과를 c에 대입, 뺄셈 연산
*= c *= a c 의 값에서 a 를 곱한 결과를 c에 대입, 곱셈 연산
/= c /= a c 의 값에서 a 를 나눈 결과를 c에 대입, 나눗셈 연산
%= c %= a c 의 값에서 a 를 나눈 나머지 결과를 c에 대입
**= c **= a c 의 값에서 a 만큼 거듭제곱한 결과를 c에 대입

비교 연산자

연산자 예 설명
> c > a c 가 a 보다 큰지 비교. 왼쪽 항(c)이 크면 참, 아니면 거짓
< c < a c 가 a 보다 작은지 비교. 왼쪽 항(c)이 작으면 참, 아니면 거짓
>= c >= a c 가 a 보다 크거나 같은지 비교. 왼쪽 항(c)이 크거나 같으면 참, 아니면 거짓
<= c <= a c 가 a 보다 작거나 같은지 비교. 왼쪽 항(c)이 작거나 같으면 참, 아니면 거짓
== c == a c 와 a 의 값이 같은지 비교. 왼쪽 항(c)과 오른쪽 항(a)의 값이 같으면 참, 아니면 거짓
=== c === a c 와 a 의 값 뿐만 아니라 자료형도 같은지 비교. 왼쪽 항(c)과 오른쪽 항(a)의 값과 자료형이 같으면 참, 아니면 거짓
!= c != a c 와 a 의 값이 다른지 비교. 왼쪽 항(c)과 오른쪽 항(a)의 값이 다르면 참, 아니면 거짓
!== c !== a c 와 a 의 값 뿐만 아니라 자료형도 다른지 비교. 왼쪽 항(c)과 오른쪽 항(a)의 값과 자료형이 다르면 참, 아니면 거짓

논리 연산자

연산자 예 설명
&& (a > 0) && (b > 0) 모두 참일 때, 결과 값이 참.
단 좌측 피연산자 식의 평가값이 거짓일 경우 우측 피연산자 식은 평가하지 않음(단락 회로 평가). 논리곱 AND
|| (a > 0) || (b > 0) 하나의 항만 참이면, 결과 값이 참.
단 좌측 피연산자 식의 평가값이 참일 경우 우측 피연산자 식은 평가하지 않음(단락 회로 평가). 논리합 OR
! !a 참인 경우 거짓, 거짓인 경우 참, 부정 NOT

증감 연산자

++ c++ c 의 값을 1 씩 증가
-- c-- c 의 값을 1 씩 감소
