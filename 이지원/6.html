산술 연산자
연산자예설명
+c = a + ba와 b의 합을 c에 저장. 덧셈 연산
-c = a - ba 와 b 의 차를 c 에 저장, 뺄셈 연산
*c = a * ba 와 b 의 곱을 c 에 저장, 곱셈 연산
**c = a ** ba 의 값을 b만큼 거듭제곱한 결과를 c에 대입
/c = a / ba 와 b 로 나눈 몫을 c 에 저장, 나눗셈 연산
%c = a % ba 와 b 로 나눈 나머지를 c 에 저장

대입 연산자
연산자예설명
+=c += ac 와 a 의 합을 c 에 저장. 덧셈 연산
-=c -= ac 에서 a 를 뺀 차를 c 에 저장, 뺄셈 연산
*=c *= ac 와 a 의 곱을 c 에 저장, 곱셈 연산
/=c /= ac 를 a 로 나눈 몫을 c 에 저장, 나눗셈 연산
%=c %= ac 를 a 로 나눈 나머지를 c 에 저장

비교 연산자
연산자예설명
==a == ba와 b의 합을 c에 저장. 덧셈 연산
!=a != ba 와 b 의 차를 c 에 저장, 뺄셈 연산
===a === ba 와 b 의 곱을 c 에 저장, 곱셈 연산
!==a !== ba 의 값을 b만큼 거듭제곱한 결과를 c에 대입
>c = a > ba 와 b 로 나눈 몫을 c 에 저장, 나눗셈 연산
>=a >= ba 와 b 로 나눈 나머지를 c 에 저장
<a < ba 와 b 로 나눈 나머지를 c 에 저장
<=a <= ba 와 b 로 나눈 나머지를 c 에 저장

논리 연산자
연산자예설명
&&a && ba와 b 둘 다 참일 때 참(and 연산)
||a || ba 와 b 둘 중에 하나만 참이면 참(or 연산)
!!aa 가 참이면 거짓, 거짓이면 참으로 바꿈(not 연산)
연산자규칙
&&좌측 피연산자 식의 평가값이 거짓이면, 우측 피연산자 식을 평가하지 않음.
||좌측 피연산자 식의 평가값이 참이면, 우측 피연산자 식을 평가하지 않음
