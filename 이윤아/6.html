산술 연산자

연산자 예 설명
+ c = a + b a와 b의 합을 c에 저장. 덧셈 연산.
- c = a - b a와 b의 차를 c에 저장. 뺄셈 연산.
* c = a * b a와 b의 곱을 c에 저장. 곱셈 연산.
/ c = a / b a를 b로 나눈 값을 c에 저장. 나눗셈 연산.
% c = a % b a를 b로 나누었을때 나머지을 c에 저장.

대입 연산자

연산자 예 설명
+= c+=a c와 a의 합을 c에 저장. 덧셈 연산.
-= c-=a c와 a의 차를 c에 저장. 뺄셈 연산.
*= c *= a c와 a의 곱을 c에 저장. 곱셈 연산
/= c /= a c를 a로 나눈 몫을 c에 저장.
%= c %= a c를 a로 나눈 뒤 나머지를 c에 저장.

비교 연산자

연산자 예 설명
== a == b a와 b의 값이 같은지 비교
!= a != b a와 b의 값이 다른지 비교
=== a === b a와 b의 값뿐만 아니라 자료형이 같은지 비교
!== a !== b a와 b의 값뿐만 아니라 자료형이 다른지 비교
> a > b a가 b보다 큰지 비교
>= a>= b a가 b보다 크거나 같은지 비교
> a> b a가 b보다 작은지 비교
>= a >= b a가 b와 같거나 작은지 비교
